# labs

### Init source code
```
yarn init -2
yarn add webpack typescript eslint
yarn dlx @yarnpkg/sdks vscode
```

Copy these files / folders:
- .vscode
- .editorconfig
- .gitattributes
- .gitignore

Enable / install recommended VSCode extensions

Copy `config/ts` folder and `.eslintignore`, `.eslintrc.js`
